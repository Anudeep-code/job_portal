# Job Portal Frontend

Frontend for the Job Portal application built using React and Vite.

## Features

* View available jobs
* Create new job postings
* Apply for jobs
* Responsive user interface
* Fast development with Vite

## Technologies Used

* React
* Vite
* JavaScript
* CSS

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```text
src/
├── App.jsx
├── main.jsx
├── App.css
└── components/
```

## Backend

This frontend communicates with the Node.js and Express backend server. Ensure the backend is running before using the application.
